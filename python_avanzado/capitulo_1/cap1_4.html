Partes funcionales del computador

(GALEANO, 2011e) Esquema de arquitectura de un computador u ordenador (Wikipedia, 2011e; Wikipedia, 2011f)

Esta sección está relacionada con lo que se conoce como arquitectura del computador que fue definida por el matemático John von Neumann, considerado como el primer científico en computación.

Es posible definir y clasificar de manera diferente las partes funcionales de un computador, ya que dependen del diseño específico. Como ejemplo podemos comentar que en los primeros computadores personales, la unidad de procesamiento matemático era un circuito integrado aparte y más tarde se integro a circuito integrado de la unidad de procesamiento (o CPU de las siglas en Control Processing Unit).

Sólo para tener una idea o modelo básico de las partes funcionales de un computador, podemos partir de la Figura (GALEANO, 2011e) donde se muestra transversalmente lo que se conoce como el BUS de conexiones y 3 partes básica que son:

CPU
Es la unidad de procesamiento central o el cerebro del computador. Esta unidad es la encargada de realizar todas las operaciones e instrucciones, controla los otros elementos de la máquina y puede tener internamente unidades de procesamiento lógico-matemático. Los computadores modernos han evolucionado a lo que se conoce como computación en paralelo, lo cual significa que pueden tener varias de estas CPU funcionando al mismo tiempo. La CPU dispone internamente de un oscilador o reloj que sincroniza la realización de cada una de sus operaciones. Mientras mayor es la frecuencia de este reloj, su velocidad de procesamiento es mayor.
MEMORY
Es la unidad de memoria y también se refieren a ella como la RAM (de las siglas en ingles Random Access Memory). Esta unidad es la que almacena la información requerida en el procesamiento y se comunica directamente con la CPU para tener acceso a sus diferentes partes o posiciones de memoria. Mientras más grande es esta memoria, mas rápido puede ser el computador, pues el tiempo de acceso a esta memoria desde la CPU es mucho menor que a otros elementos del computador.
INPUT AND OUTPUT
Es la unidad que controla otros elementos para entrada y salida de información al computador. Entre las unidades de salida podemos enumerar las unidades de almacenamiento, tales como discos duros internos o externos. En algunos casos los discos duros pueden estar directamente conectados al sistema conexiones (SYSTEM BUS). Las impresoras, las pantallas, los parlantes de sonido, las memorias USB (de las siglas en ingles Universal Serial Bus) son también ejemplos comunes de unidades de salida y/o entrada. Los puertos de comunicación que permiten la iteracción con las redes de computadores son parte de estos elementos de entrada y salida. Ellos realizan las funciones de intermediación entre lo que maneja el computador y el canal físico de comunicación según el caso. Ejemplo de estos puertos son el conocido BLUTOOTH, el Modem-fax, etc.
SYSTEM BUS
es el punto común de conexión de las diferentes unidades y es la autopista de comunicación entre todos los elementos. En términos de vías, podemos decir que se distinguen 3 carriles, uno para traer y llevar datos, otro que para instrucciones o comando y el carril de direcciones de memoria. Una mayor velocidad de funcionamiento se logra con un número de BIT mayor.

La temática de arquitectura de computadores es toda una ciencia y los detalles técnicos se exploran en cursos avanzados de circuitos digitales y circuitos integrados. Aquí la idea es tener un modelo básico para comprender la lógica de funcionamiento de estos complejos aparatos.
