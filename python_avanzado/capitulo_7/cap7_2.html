Comandos de entrada de datos

El teclado en el ordenador es el primer elemento disponible para entrada de información y los lenguajes de programación disponen de comando para esperar que el usuario ingrese algún dato antes de continuar el proceso.

En Python se dispone de dos funciones para la entrada de información desde el teclado: los comandos raw_input() e input(). El comando raw_input() recibe la información siempre como una cadena de caracteres directamente desde el teclado y hace la pausa en el programa hasta cuando se ingresa totalmente la información terminada por la tecla de entrar ( 'enter'). El argumento de entrada en la función es un comentario que aparecerá en la pantalla antes del espacio de ingreso de datos. Veamos un ejemplo de su aplicación en el interface de Python (Python shell):


>>>dato1 = raw_input('Escriba un dato aqui: ')
Escriba un dato aquí: TECLADO
>>>dato1
'TECLADO'
>>>type(dato1)
<type 'str'>
>>>

La primera línea define que se asignará en la variable dato1 lo que se escriba desde el teclado y aparecerá primero un mensaje diciendo Escriba un datos aquí: para esperar la entrada del dato. Al escribir la palabra TECLADO, ésta se guardará en la variable tipo alfanumérica ( 'str' ) denominada dato1 como se muestra al usar la función type().

Es importante recalcar que aún cuando se solicita un número, la función siempre interpretará la entrada como caracteres, tal como se muestra en el siguiente ejemplo:


>>>dato2 = raw_input('Escriba un numero: ')
Escriba un numero: 6.2827
>>>dato2
'6.2827'
>>>type(dato2)
<type 'str'>
>>>dato2n=float(dato2)
>>>type(dato2n)
<type 'float'>
>>>

La variable dato2 es una cadena de caracteres y no un número para realizar operaciones matemáticas. Si se requiere un número será necesario transformar la variable dato2, usando float() y obtener una nueva variable numérica dato2n como lo muestran las últimas líneas.

Una opción alternativa que dispone Python es el comando input(), el cual interpreta la información de entrada a un tipo de variable adecuado. Veamos el siguiente ejemplo:


>>>dato3 = input('Entre un numero entero : ')
Entre un numero entero : 34
>>>dato3
34
>>>type(dato3)
<type 'int'>
>>>dato4 = input('Entre un numero real : ')
Entre un numero real : 4.3421
>>>dato4
4.3421
>>>type(dato4)
<type 'float'>
>>>

Observe que dato3 y dato4 son variables de diferente tipo, la primera es de tipo entero ('int') y la segunda de tipo real ('float'). Este proceso de interpretación lo realiza directamente el comando antes de asignar el dato a la variable.

El uso de la función input() requiere especial atención cuando se requiere usar información de tipo alfanumérico o lógico, ya que puede presentar un error al ingreso de la información. Veamos estos dos casos para ilustrar el resultado:


>>>dato5 = input('Escriba Si o No : ')
Escriba Si o No : si

Traceback (most recent call last): 
  File "<pyshell#19>", line 1, in <module>
  	dato5 = input('Escriba Si o No : ')
  File "<string>", line 1, in <module>
NameError: name 'si' is not defined
>>>dato5 = input('Escriba Si o No : ')
Escriba Si o No : 'si'
>>>dato5
'si'
>>>type(dato5)
<type 'str'>
>>>

Aquí en el primer intento aparece un mensaje de error informando que el nombre si no está definido (“name 'si' is not defined”), ya que el comando trató de interpretar si como un objeto y no encontró ninguna referencia de esa palabra. En la segunda opción se escribió entre comillas la palabra si ('si') y se interpretó como un dato tipo caracteres o alfanumérico.


>>>dato6 = input('Es true or false ? ')
Es true or false ? true

Traceback (most recent call last):
  File "<pyshell#23>", line 1, in <module>
	dato6 = input('Es true or false ? ')
  File "<string>", line 1, in <module>
NameError: name 'true' is not defined
>>>dato6 = input('Es true or false ? ')
Es true or false ? True
>>>dato6
True
>>>type(dato6)
<type 'bool'>
>>>

Aquí también se presenta el mismo mensaje de error ya que el dato lógico True requiere ser escrito con una mayúscula inicial.
